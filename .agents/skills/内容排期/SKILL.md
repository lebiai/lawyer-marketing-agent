# 内容排期

## 工作流

```
Step 1: 询问用户创作节奏（均衡/高频/聚焦）
Step 2: 调用 generate_calendar 生成排期
Step 3: 输出日历 + 可选结合热点追踪优化
Step 4: 调用 log_conversation
```

## 可用模板

| 模板 | 频率 | 适合场景 |
|------|------|---------|
| 均衡型 | 每周3篇 | 大部分创作者 |
| 高频型 | 每周5篇 | 全职创作者 |
| 聚焦型 | 每周2篇 | 兼职/新手 |

## 输出格式

```
════════════════════════════════
📅 内容排期日历
════════════════════════════════

📌 模板：[均衡型/高频型/聚焦型]
📌 周期：4 周
📌 总计：X 篇内容

第 1 周
  [日期] [平台] [内容类型] — ⏳ 待创作
  [日期] [平台] [内容类型] — ⏳ 待创作
  ...

第 2 周
  ...

💡 提示：可结合热点追踪优化选题
════════════════════════════════
```
