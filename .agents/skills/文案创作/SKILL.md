# 文案创作

## 工作流

```
Step 1: 调用 generate_outline 获取平台大纲 + KPI 参考
Step 2: 搜索知识库 → 匹配风格样本
Step 3: 生成 3 个方向让用户选择
Step 4: 用户选定方向
Step 5: 按平台方法论展开写全文
Step 6: 调用 store_knowledge(my_articles)
Step 7: 调用 log_conversation
```

## 平台创作方法论

### 📕 小红书

**标题公式：** 15–22 字 = 数字/利益/痛点 + emoji + 悬念/关键词
**正文：** 300–800 字，短句分段，每段≤3 行，emoji 分隔
**结构：**
| 段落 | 要求 |
|------|------|
| 开头 | 直接给好处/痛点，第一人称 |
| 中间 | 分点讲干货/真实体验 |
| 结尾 | 互动引导 |
**语气：** 第一人称"我/姐妹们"，种草分享感

### 🎵 抖音

**黄金 60 秒结构：**
| 时段 | 内容 |
|------|------|
| 0–3 秒 | **钩子**：反常识/提问/冲突/结果前置 |
| 3–30 秒 | **展开**：痛点→原因→方案，短句≤15字 |
| 30–50 秒 | **亮点**：增量信息/反转 |
| 50–60 秒 | **引导**：点赞/评论/关注 |
**视频长度：** 新手 15–30 秒，成熟 60–120 秒
**字幕：** 关键句放大，关键词重复 2 次

### 📰 公众号

**标题公式：** 20–30 字 = 利益/冲突/悬念/反差/热点 + 价值承诺
**正文：** 1200–2500 字，段落 2–4 行，重点加粗
**结构：** 导语金句 → 提出问题 → 分层论述(案例/数据) → 总结观点 → 互动引导
**排版：** 行间距 1.5–1.75，配图 3–5 张，段落留白
**推送：** 早 7:30 / 午 12:00 / 晚 20:00

## 参考 KPI

| 平台 | 核心指标 | 优质标准 |
|------|---------|---------|
| 小红书 | 收藏率 > 深度阅读 > 互动 | 收藏率 > 8% |
| 抖音 | 完播率 > 3秒留存 > 互动 | 完播率 ≥ 45% |
| 公众号 | 打开率 > 读完率 > 在看 | 打开率 ≥ 12% |
