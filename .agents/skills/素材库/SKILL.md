# 素材库管理

## 工作流

### 存储素材
```
Step 1: 用户提供素材名称、分类、描述、标签
Step 2: 调用 store_asset 存入 personal_notes
Step 3: 确认存储结果
Step 4: 调用 log_conversation
```

### 搜索素材
```
Step 1: 用户提供关键词或分类
Step 2: 调用 search_assets 搜索
Step 3: 输出匹配的素材列表
Step 4: 调用 log_conversation
```

## 可用分类

| 分类 | 说明 |
|------|------|
| image_template | 图片模板 |
| video_clip | 视频片段 |
| brand_color | 品牌色板 |
| font_style | 字体风格 |
| icon_set | 图标集 |
| music_track | 背景音乐 |
| voice_sample | 语音样本 |
