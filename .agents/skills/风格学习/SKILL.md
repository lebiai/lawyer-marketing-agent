# 风格学习

当用户提供文案样本时执行。

## 工作流

```
Step 1: 调用 analyze_style 自动提取四层维度特征
Step 2: 展示风格分析报告
Step 3: 计算 embedding → 存入 content_samples（附带结构化 features）
Step 4: 调用 log_conversation
```

## 四层分析维度

### 1️⃣ 定位与基调（气质底色）
- 目标受众：写给谁看？
- 核心目的：种草/共鸣/信任/信息/话题
- 情感基调：理性客观/温暖治愈/犀利直白/幽默俏皮/高级克制/热血有力/文艺感性
- 语气立场：平等对话/权威指导/朋友分享/官方正式/自嘲玩梗

### 2️⃣ 语言与文字（风格载体）
- 句式特征：平均句长、长短句分布、疑问/祈使句比例
- 用词风格：口语化/书面化/专业术语/网络热词/古典意象
- 篇幅节奏：一句话/短段落/分段清晰/长文多段
- 人称使用：第一人称/第二人称/第三人称

### 3️⃣ 表达与修辞（感染力）
- 修辞手法：比喻/对比/排比/设问/引用/数字具象化/反问/夸张
- 叙事逻辑：痛点切入→方案/故事线/观点+论证/直接陈述
- 信息侧重：功能/利益/情感/身份/信息

### 4️⃣ 传播与适配（场景感）
- 适配平台：朋友圈/短视频/小红书/公众号
- 独特符号：固定句式/口头禅/标志性意象
