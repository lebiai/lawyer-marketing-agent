# 热点追踪

## 工作流

```
Step 0: 调用 check_search_tools 检测可用搜索工具
Step 1: 调用 get_search_strategy 获取搜索策略（5个维度）
Step 2: 使用可用工具对每个维度的关键词逐一搜索
         首选 agent-reach → 降级 anysearch → 兜底 web_search
Step 3: 调用 analyze_topics 分析热度 + 去重 + 平台匹配
Step 4: 输出热点榜单（含选题建议 + 内容角度）
Step 5: 调用 store_knowledge(hot_topics) 存储去重后的话题
Step 6: 调用 log_conversation
```

## 搜索工具

| 工具 | 能力 | 优先级 |
|------|------|--------|
| agent-reach | 多平台搜索（小红书/微博/知乎/B站） | 首选 |
| anysearch | 通用网页搜索 | 降级 |
| web_search | Codex 内置搜索 | 兜底 |

## 搜索维度

| 维度 | 搜索方向 | 内容类型 |
|------|---------|---------|
| 政策法规 | 新规出台、司法动态 | 解读分析 |
| 热点案件 | 重大案件、名人纠纷 | 案例评析 |
| 行业趋势 | 行业发展、法律科技 | 行业观察 |
| 民生普法 | 劳动仲裁、婚姻财产 | 普法科普 |
| 求职就业 | 法考、律师招聘 | 职业指导 |

## 输出格式

```
════════════════════════════════
🔥 律师行业热点追踪
════════════════════════════════

🔧 搜索工具：[agent-reach / anysearch / web_search]
📊 统计：共搜索 X 条，筛选出 X 个新热点

🔥 [热度] [标题]
  📌 [简介]
  📂 类型：[解读分析/案例评析/...]
  📱 推荐平台：[小红书/抖音/公众号]
  💡 切入角度：
    1. [角度1]
    2. [角度2]
    3. [角度3]

🔥 [热度] [标题]
  ...

════════════════════════════════
```
