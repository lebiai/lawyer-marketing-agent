# 竞品账号分析

对标账号的内容策略与风格分析，输出结构化报告。

## 工作流

```
Step 1: 用户提供目标账号名称/平台/至少3篇文案样本
Step 2: 调用 analyze_account 进行四层风格+内容策略+互动+话题分析
Step 3: 展示完整分析报告
Step 4: 调用 store_knowledge(competitor_analysis) 归档
Step 5: 调用 log_conversation
```

## 六维分析框架

与风格学习共享四层风格维度，额外增加两层：

### 1️⃣ 定位与基调
- 目标受众：写给谁看？（大众/职场/年轻人/专业人士/特定场景）
- 情感基调：理性客观/治愈/犀利/幽默/克制/热血/文艺
- 语气立场：平等对话/权威指导/朋友分享/官方正式/自嘲玩梗

### 2️⃣ 语言与文字
- 句式特征：平均句长、短中长句分布、疑问/祈使句比例
- 用词风格：口语化/书面化/专业术语/网络热词
- 篇幅节奏：短段落/分段清晰/长文多段
- 人称使用：第一/第二/第三人称偏好

### 3️⃣ 表达与修辞
- 修辞手法：比喻/对比/排比/设问/反问/引用/夸张
- 叙事逻辑：痛点切入→方案/场景→共鸣/故事线/观点+论证/直接陈述
- 信息侧重：功能/利益/情感/身份/价值观

### 4️⃣ 传播与适配
- 适配平台：小红书/抖音/公众号
- 风格标签：一句话总结风格特征

### 5️⃣ 内容策略
- 内容类型分布（科普/案例/观点/故事/热点/工具）
- 篇幅偏好（短/中/长）
- 更新频率判断

### 6️⃣ 互动表现 + 高频话题
- 平均互动数据（点赞/评论/收藏/转发）
- 高互动内容特征
- 话题领域分布与切入角度建议

  ▸ 目标受众：[受众]
  ▸ 情感基调：[基调]

【✏️ 语言与文字】

【📱 传播与适配】

【📋 内容策略】
