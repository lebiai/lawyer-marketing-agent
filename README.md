> **你给一个竞品账号链接，它帮你拆透这个号；你给一个选题方向，它按竞品风格写出全平台文案；热点一出，它帮你追、帮你想、帮你写。**

乐彼AI 不是一个「工具」，是一个**在本地运行的多智能体内容团队**——分析员、文案写手、热点编辑、品牌管家、素材管理员……都在你的终端里，随时等你指挥。

## 🎯 它能解决你的什么痛苦？

### 「每天打开手机，刷了俩小时同行，啥也没记住」
→ **竞品账号分析**：给链接，15 分钟出一份完整拆解报告——人设公式、爆款规律、受众需求、复刻策略。不是给你看数据，是告诉你「该怎么做」。

### 「写一篇小红书要憋一上午，还总是凉」
→ **风格学习 + 文案创作**：先学对标账号怎么写的，再用它的风格帮你写。句长分布、语气、修辞、开头方式……连 emoji 密度都模仿到位。

### 「热点来了别人都发了，我才开始想选题」
→ **热点追踪**：自动扫描多个来源，去重+热度评分，直接给出选题建议。你只需要说「开始写」。

### 「一篇稿子改三个平台，格式调半天」
→ **多平台适配**：平台规则可配置不写死。一键从「小红书体」转成「抖音口播稿」再转「公众号长文」，结构对、语气对、长度对。

### 「素材散得到处都是，下次找不到了」
→ **素材库**：全部可检索，自然语言搜「离婚财产分割那个案例」一秒命中。

### 🎯 竞品账号分析 — 最亮眼的能力

给一个小红书/抖音账号链接，自动完成：

```
输入一个链接 →
     │
     ▼
1️⃣ 采集内容：自动获取该账号的公开信息
   ├─ 用户基本信息（昵称、头像、简介、粉丝数等）
   ├─ 作品列表（含互动数据）
   ├─ 每篇作品的详细内容（正文、图片、视频、标签）
   └─ 评论区（分析受众在说什么）
     │
     ▼
2️⃣ AI 深度分析：
   ├─ 人设公式：这个号是怎么定位的？标签+简介+内容线索
   ├─ 复刻策略：如果我想做类似的号，该怎么入手？
   ├─ 受众画像：关注这个号的人有什么需求？
   ├─ 爆款规律：哪些内容数据好？为什么？
   ├─ 内容策略：更新频次、主题分布、话题标签策略
   └─ 竞品差异：我和他比，机会在哪里？
     │
     ▼
3️⃣ 输出结果：
   ├─ 完整 HTML 报告（浏览器直接打开）
   ├─ 风格特征自动入库，下次写文案调用
   └─ 分析结论可检索，之后再问直接引用
```

**内容本地全量存储**：每个分析的账号，图片、视频、评论、分析报告完整保存在本地文件夹，方便随时翻看。

### ✍️ 文案创作 — 不是模板堆砌

**流程**：你给选题 → AI 出 3 个方向 → 选定 → 按平台规则+风格 → 生成全文

| 平台 | 创作方法 |
|------|---------|
| 📕 **小红书** | 数字/利益+emoji+悬念标题公式，300-800 字分段干货 |
| 🎵 **抖音口播** | 0-3 秒钩子→展开→亮点→引导，短句紧凑不拖沓 |
| 📰 **公众号** | 利益/冲突/悬念标题，1200-2500 字深度长文 |

### 🧠 风格学习 — 系统性拆解对手

不只是看「语气」，是系统分析：

| 层级 | 我们分析什么 |
|------|------------|
| 定位与基调 | 写给谁？什么目的？什么情绪？什么姿态？ |
| 语言与文字 | 句长分布、词汇风格、人称使用、篇幅节奏 |
| 表达与修辞 | 用了什么修辞手法？怎么展开内容？强调什么？ |
| 传播与适配 | 适合发什么平台？一句话风格标签是什么？ |

## 🤔 常见问题

### 这个和直接用 ChatGPT 有什么区别？

乐彼AI 跑在 Codex 里，有**持久化的本地知识库**。你用 ChatGPT 问完就结束了，但乐彼AI 会：
- 把分析过的账号特征存下来，下次直接复用
- 把学过的风格存下来，写文案时可以选「按这个风格」
- 把素材存下来，能自然语言检索
- 把热点追踪历史存下来，能看趋势

**简单说：ChatGPT 用完即走，乐彼AI 越用越聪明。**

### 一定要联网吗？

核心写作功能（风格学习、文案创作、品牌检查）不依赖网络。竞品分析和热点追踪需要联网采集数据。所有数据都存在你本地。

### 可以做非律师行业吗？

可以。架构完全通用，种子数据预置了律师行业内容。更换行业只需更新模板数据，随时可以切换。

### 数据安全吗？

所有数据都存在你本地。分析结果、知识库、素材——都在你自己的电脑上。
